## Current State Analysis

### 1. Transcription Architecture

The project has **TWO separate Gemini transcription implementations**:

#### Frontend (Client-Side) - PRIMARY/USED
- **Location**: `frontend/src/App.tsx`
- **SDK**: `@google/genai` (JavaScript/TypeScript GenAI SDK) v1.29.0
- **Model Options**: 
  - `gemini-3.1-pro-preview` (default)

- **Function**: `transcribeAudio()` (lines 413-475)

### 3. Key Issues Identified

#### Issue 1: Invalid API Keys (CRITICAL)
- Both `.env` files contain placeholder text, NOT valid Gemini API keys
- **Impact**: Transcription will fail with authentication errors
- **Frontend error**: `GoogleGenAI` will reject the placeholder key
- **Backend error**: `GeminiService.__init__()` will still initialize but API calls will fail

#### Issue 2: SDK & Model Mismatch
- Frontend and backend use different Google GenAI SDKs entirely
- Frontend uses newer `@google/genai` with Gemini 3.x models
- Backend uses older `google-generativeai` with Gemini 1.5 model
- **Impact**: Inconsistent behavior if both are used; backend endpoints are essentially orphaned

#### Issue 3: Backend Module Initialization
- `server/routes/captions.py` line 10: `gemini = GeminiService()` at module level
- **Impact**: Server will crash on startup if `GEMINI_API_KEY` is missing/invalid because the GeminiService constructor raises `ValueError` when API key is not configured
- However, the key IS configured (placeholder), so initialization succeeds but API calls will fail

#### Issue 4: Frontend Model Selection vs Backend Model
- Frontend UI lets users select from 3 Gemini 3.x models
- Backend hardcodes `gemini-1.5-pro`
- No way to pass model preference from frontend to backend (if they were connected)

#### Step 2.4: Create Vertex AI Service

**Create `server/services/vertex_service.py`** (new file):
```python
import os
import json
import base64
from google.cloud import aiplatform
from google.oauth2 import service_account
from typing import Dict, Any

class VertexService:
    def __init__(self):
        self.project_id = os.environ.get('GCP_PROJECT_ID')
        self.region = os.environ.get('GCP_REGION', 'us-central1')
        creds_path = os.environ.get('GCP_CREDENTIALS_PATH')
        
        if not self.project_id or not creds_path:
            raise ValueError("GCP_PROJECT_ID and GCP_CREDENTIALS_PATH must be set")
        
        # Initialize Vertex AI
        credentials = service_account.Credentials.from_service_account_file(
            creds_path
        )
        aiplatform.init(
            project=self.project_id,
            location=self.region,
            credentials=credentials
        )
    
    def transcribe(self, audio_path: str) -> Dict[str, Any]:
        """Transcribe audio using Gemini via Vertex AI"""
        # Read audio file
        with open(audio_path, 'rb') as f:
            audio_bytes = f.read()
        
        # Convert to base64 for Vertex AI
        audio_b64 = base64.b64encode(audio_bytes).decode('utf-8')
        
        # Use Gemini 1.5 Pro or 2.0 Flash via Vertex AI
        model = aiplatform.GenerativeModel(
            model_name="gemini-1.5-pro-preview-0409"  # or "gemini-2.0-flash-exp"
        )
        
        prompt = """You are an expert audio transcriptionist. Analyze this audio file.

Return ONLY valid JSON with these exact fields:
{
  "language": "Detected Language (English, Hindi, Hinglish, etc.)",
  "transcript": "Full transcript text"
}

Rules:
- Detect language accurately
- For Hinglish: Hindi in Devanagari, English in Latin
- Preserve all spoken content
- Include natural pauses with "..." or line breaks"""
        
        try:
            response = model.generate_content(
                contents=[
                    {
                        "role": "user",
                        "parts": [
                            {"text": prompt},
                            {
                                "inline_data": {
                                    "mime_type": "audio/mpeg",  # adjust based on file type
                                    "data": audio_b64
                                }
                            }
                        ]
                    }
                ],
                generation_config={"response_mime_type": "application/json"}
            )
            
            text = response.text.strip()
            # Clean markdown code blocks if present
            if text.startswith('```json'):
                text = text[7:]
            if text.startswith('```'):
                text = text[3:]
            if text.endswith('```'):
                text = text[:-3]
            
            return json.loads(text.strip())
        except Exception as e:
            return {
                'language': 'Unknown',
                'transcript': '',
                'error': str(e)
            }
    
    def align_transcript(self, audio_path: str, transcript: str) -> Dict[str, Any]:
        """Align transcript segments with audio timestamps"""
        with open(audio_path, 'rb') as f:
            audio_bytes = f.read()
        audio_b64 = base64.b64encode(audio_bytes).decode('utf-8')
        
        prompt = f"""Given this audio and transcript, create precise caption segments.

TRANSCRIPT: {transcript}

Create 2-6 word segments with accurate start/end timestamps.

Return ONLY JSON:
{{"segments": [{{"start": 0.0, "end": 2.5, "text": "Segment text"}}]}}"""
        
        model = aiplatform.GenerativeModel(
            model_name="gemini-1.5-pro-preview-0409"
        )
        
        response = model.generate_content(
            contents=[
                {
                    "role": "user",
                    "parts": [
                        {"text": prompt},
                        {
                            "inline_data": {
                                "mime_type": "audio/mpeg",
                                "data": audio_b64
                            }
                        }
                    ]
                }
            ],
            generation_config={"response_mime_type": "application/json"}
        )
        
        try:
            text = response.text.strip()
            if text.startswith('```json'):
                text = text[7:]
            if text.startswith('```'):
                text = text[3:]
            if text.endswith('```'):
                text = text[:-3]
            
            result = json.loads(text.strip())
            segments = result.get('segments', [])
            
            # Ensure timestamps are floats and sort
            for seg in segments:
                seg['start'] = float(seg.get('start', 0))
                seg['end'] = float(seg.get('end', 0))
            segments.sort(key=lambda x: x['start'])
            
            return {'segments': segments}
        except Exception:
            return {'segments': []}
    
    def sync_segments(self, audio_path: str, current_segments: list) -> Dict[str, Any]:
        """Re-sync existing segments to audio"""
        with open(audio_path, 'rb') as f:
            audio_bytes = f.read()
        audio_b64 = base64.b64encode(audio_bytes).decode('utf-8')
        
        segments_json = json.dumps(current_segments, ensure_ascii=False)
        
        prompt = f"""Re-sync these caption segments to match the audio precisely.
Keep text unchanged, adjust timestamps only.

CURRENT SEGMENTS:
{segments_json}

Return JSON with updated timestamps:
{{"segments": [{{"start": 1.234, "end": 3.456, "text": "Original text"}}]}}"""
        
        model = aiplatform.GenerativeModel(
            model_name="gemini-1.5-pro-preview-0409"
        )
        
        response = model.generate_content(
            contents=[
                {
                    "role": "user",
                    "parts": [
                        {"text": prompt},
                        {
                            "inline_data": {
                                "mime_type": "audio/mpeg",
                                "data": audio_b64
                            }
                        }
                    ]
                }
            ],
            generation_config={"response_mime_type": "application/json"}
        )
        
        try:
            text = response.text.strip()
            if text.startswith('```json'):
                text = text[7:]
            if text.startswith('```'):
                text = text[3:]
            if text.endswith('```'):
                text = text[:-3]
            
            result = json.loads(text.strip())
            segments = result.get('segments', [])
            
            for seg in segments:
                seg['start'] = float(seg.get('start', 0))
                seg['end'] = float(seg.get('end', 0))
            segments.sort(key=lambda x: x['start'])
            
            return {'segments': segments}
        except Exception:
            return {'segments': current_segments}
```

### PHASE 6: Deployment Considerations

#### Step 6.1: Secure Credential Management
For production deployment (Cloud Run, GKE, etc.):
- Use GCP Workload Identity (no JSON file needed)
- Or use Secret Manager to store service account JSON
- Or set `GOOGLE_APPLICATION_CREDENTIALS` env var to JSON path

#### Step 6.2: Audio File Size Limits
Vertex AI has request limits:
- Audio files: max ~20MB for Gemini 1.5 Pro (depends on model)
- Long audio (>10 min): need to chunk and transcribe in parts

Your current `MAX_CONTENT_LENGTH = 50 * 1024 * 1024` in config.py is fine for upload, but Vertex may reject >20MB audio. Consider:
- Reject large files upfront with validation
- Or implement chunked transcription (split audio, transcribe each, combine)

#### Step 6.3: Cost Management
Vertex AI pricing per minute of audio:
- Gemini 1.5 Pro: ~$0.012-$0.036 per minute (input+output)
- Monitor usage in GCP Console → Billing
